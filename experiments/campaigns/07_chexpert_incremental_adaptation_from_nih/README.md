# 07 CheXpert Incremental Adaptation From NIH

This campaign contains the first incremental `NIH -> CheXpert` adaptation experiments that warm-start from the strong `NIH` source head instead of training a fresh target-only head.

Included here:

- merged NIH + CheXpert embedding-root views for adaptation
- warm-started head-adaptation runs on `CheXpert`
- post-adaptation evaluation on both `NIH test` and `CheXpert test`

Primary range:

- `exp0074` to `exp0075`
