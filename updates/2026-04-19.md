# Supervisor-Facing Update: Oracle, Transfer, Naive Continual, `LwF`, and `MAS`

Prepared on `April 19, 2026 (UTC)`.

## Executive summary

The continual-learning pilot is now complete in the requested order:

1. independent single-domain oracle baselines
2. `NIH` cross-domain transfer baseline
3. naive continual baseline
4. sequential rehearsal-free `LwF`
5. sequential rehearsal-free `MAS`

This is a head-only, frozen-embedding study on `CXR Foundation` features. The only trainable component is a `768 -> 7` linear multilabel head with `5.4k` parameters and about `10.8k` FLOPs per image.

## 1. Independent single-domain oracle baselines

- `NIH` oracle: `0.8461`
- `CheXpert` oracle: `0.7703`
- `MIMIC` oracle: `0.7619`

These are the current same-domain ceilings of the embedding space under a linear head.

## 2. `NIH` cross-domain transfer baseline

- `NIH -> CheXpert`: `0.8486`
- `NIH -> MIMIC`: `0.7370`

This remains asymmetric:

- `NIH -> CheXpert` is already stronger than the `CheXpert` oracle
- `NIH -> MIMIC` is still below the `MIMIC` oracle

So `CheXpert` is already highly aligned with the `NIH` source model, while `MIMIC` still has a real transfer gap.

## 3. Naive continual baseline

The baseline chain is:

1. `NIH` train
2. fine-tune on `CheXpert`
3. fine-tune on `MIMIC`

The previously missing Stage-B-on-`MIMIC` readout has now been backfilled from the saved Stage-B checkpoint with no retraining, so the Stage-B `3 x 3` matrix is complete.

| Stage | NIH test | CheXpert test | MIMIC test |
| --- | ---: | ---: | ---: |
| Stage A: `NIH` source | `0.8461` | `0.8486` | `0.7370` |
| Stage B: after `CheXpert` | `0.8210` | `0.7882` | `0.7638` |
| Stage C: after `MIMIC` | `0.8221` | `0.8135` | `0.7770` |

Main takeaways:

- the main forgetting event still happens at the `NIH -> CheXpert` step
- `NIH` drops by `-0.0251` after Stage B
- final `NIH` remains `-0.0240` below the original source model
- `MIMIC` does improve under the naive chain, from `0.7370` to `0.7770`

## 4. Sequential rehearsal-free `LwF`

This is the strict no-replay version:

- Stage B distills teacher `A` on `CheXpert` inputs only
- Stage C distills teacher `B` on `MIMIC` inputs only

Best test setting:

- `alpha=1.0`, `T=8`
- final test AUROC: `NIH 0.8316`, `CheXpert 0.8230`, `MIMIC 0.7768`

Relative to the naive final stage:

- `NIH +0.0095`
- `CheXpert +0.0095`
- `MIMIC -0.0002`

Interpretation:

- `LwF` is a real improvement over naive continual training
- it mainly improves retention and `CheXpert`
- it does not materially improve the final `MIMIC` result

## 5. Sequential rehearsal-free `MAS`

This is strict saved-state online `MAS`:

- `omega_NIH` is computed after Stage A
- `omega_CheXpert` is computed after Stage B and accumulated
- Stage C uses only the saved accumulated state
- no prior-stage data are reopened

Two points matter most:

Balanced point:

- `lambda=0.3`
- final test AUROC: `NIH 0.8341`, `CheXpert 0.8225`, `MIMIC 0.7791`
- vs naive final stage: `+0.0120`, `+0.0090`, `+0.0020`

Retention-first point:

- `lambda=30`
- final test AUROC: `NIH 0.8465`, `CheXpert 0.8475`, `MIMIC 0.7444`
- vs naive final stage: `+0.0244`, `+0.0340`, `-0.0326`

Interpretation:

- `MAS` produces the clearest retention-versus-adaptation tradeoff curve
- `lambda=0.3` is the best practical rehearsal-free baseline in this pilot
- high-`lambda` `MAS` can almost fully preserve the original `NIH` source model and the original `NIH -> CheXpert` transfer behavior, but it gives up part of the `MIMIC` gain

## Supervisor-facing readout

1. The strongest overall rehearsal-free baseline in this pilot is `MAS`, not `LwF`.
2. `LwF` helps, but the gains are modest and mostly retention-oriented.
3. `MAS lambda=0.3` is the best balanced point because it improves all three final test domains over the naive chain.
4. `MAS lambda=30` is the clean retention-first result because it nearly restores the original Stage-A `NIH` and `NIH -> CheXpert` behavior.
5. The main scientific story is still that `CheXpert` is already highly aligned with the `NIH` source head, while `MIMIC` is the domain where adaptation is genuinely useful.
